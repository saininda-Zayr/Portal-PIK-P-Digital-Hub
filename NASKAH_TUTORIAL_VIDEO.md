# NASKAH VIDEO TUTORIAL SIM-ASN
## Bidang PIK-P Kabupaten Polewali Mandar

### Video 1: Panduan Pengunggahan Dokumen (Durasi Estimasi: 3 Menit)
**Visual**: Tampilkan Dashboard Utama, lalu klik tombol "Tambah Dokumen".
**Narasi**: 
"Halo rekan-rekan Bidang PIK-P. Pada tutorial pertama ini, kita akan belajar cara mengunggah dokumen dengan standar penamaan otomatis. Pertama, klik tombol tambah di pojok kanan atas. Pilih bulan dan tahun dokumen. Kemudian, pilih kode kegiatan yang sesuai—misalnya Data LHKPN. Pada kolom nama file, masukkan deskripsi singkat dengan huruf kapital. Terakhir, seret file Anda ke area box atau klik untuk memilih file. Klik SIMPAN, dan sistem akan otomatis mengubah nama file Anda sesuai standar di Google Drive."

### Video 2: Sinkronisasi Google Drive (Durasi Estimasi: 2 Menit)
**Visual**: Klik menu salah satu kategori, lalu sorot tombol "Sinkron Drive".
**Narasi**:
"Seringkali kita memiliki file lama di Drive yang namanya belum rapi. Melalui fitur Sinkron Drive, sistem akan memindai folder Anda secara otomatis. Klik tombol 'Sinkron Drive' di bagian atas. Tunggu prosesnya, dan aplikasi akan merapikan nama file lama Anda serta memasukkannya ke database aplikasi. Ini memastikan tidak ada lagi 'data silo' atau data yang tersembunyi di folder pribadi."

### Video 3: Navigasi dan Pencarian (Durasi Estimasi: 2 Menit)
**Visual**: Gunakan kolom search dan filter tahun di Dashboard.
**Narasi**:
"Mencari dokumen di SIM-ASN sangatlah mudah. Anda bisa menggunakan filter tahun di sebelah kanan untuk menyaring data per periode. Atau, gunakan kolom pencarian untuk mengetikkan kata kunci tertentu. Klik ikon mata untuk melihat detail dokumen atau membuka file aslinya di Google Drive. Dengan sistem ini, seluruh pekerjaan bidang terdokumentasi dengan transparan."
